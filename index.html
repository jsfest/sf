Evolution of Experience

Sunday, March 9th

The user experience of the web has continued to evolve, as has the experience of developing for it. We write applications faster, for more sizes and device, and with wider variety of tools than ever before. Learn about the latest tools and the successful experiences of those that have adopted them.

Topics: npm, browserify, backbone

Scaling Up and Down

Monday, March 10th

Can the tools we use to build a website in half a day also help us add features to an app that deploys to millions? Learn about tools and tricks to help you start fast and scale big in the front-end to the back-end.

Topics: node.js, WebSockets, WebRTC, bootstrap
